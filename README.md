# alfred-workflows
a collection of alfred workflows

## Installation of workflows

If you want to install a single workflow, click `View Raw` on the file you want download the .alfredworkflow file and double click it to open it in Alfred.

## Workflows

### yt-clip

Clip -15/+15 seconds from a YouTube video at a timestamp.

### PM Obsidian

Append to Resources or Kanban file in an Obsidian vault. [Learn more about this here](https://www.bramadams.dev/202305121544/#code)

### Important Apps Switcher

An example of how to use Alfred to set up hotkeys b/w different apps
